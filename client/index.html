AI Pipeline

How It Works

Five-step intelligent processing pipeline

01
Voice / Text Input
User speaks into microphone or types a natural language command. Web Speech API transcribes audio in real-time.
Web Speech API · WebRTC
02
Speech Recognition
Raw audio is converted to text. In a production system, OpenAI Whisper provides industry-leading accuracy across accents.
Whisper / Web Speech API
03
NLP Entity Extraction
Claude AI uses Named Entity Recognition to extract amount, item name, and semantic context from the transcript.
Claude AI · NER · spaCy
04
Category Classification
ML classifier maps extracted items to expense categories using trained intent detection and rule-based fallback.
scikit-learn · Intent Classification
05
Storage & Visualization
Structured expense is stored in the database. Dashboard charts update automatically with the new data point.
SQLite · Chart.js · FastAPI

📖 Project Info

About Spendly

A complete AIML submission project demonstrating voice interfaces, NLP, and ML classification.

Project Overview

Spendly is an AI-powered expense tracking system where users record daily expenses using natural voice commands instead of manually entering data. The system converts spoken input into text, extracts important information (amount, category, item), and stores it in a database.

This project demonstrates the complete integration of Speech Recognition, Natural Language Processing, Machine Learning classification, and real-time data visualization in a single cohesive application.

AI/ML Technologies

OpenAI Whisper
Production-grade speech-to-text with multilingual support and noise robustness.
Speech Recognition
Claude AI
NLP entity extraction for amount, item and category from natural language text.
NLP · NER
spaCy / NLTK
Named Entity Recognition and tokenization for structured data extraction pipeline.
NER · Tokenization
scikit-learn
ML classifier trained on expense sentences for automatic category prediction.
ML Classification
FastAPI
Python backend serving the AI pipeline with async REST endpoints.
Backend · Python
SQLite / MongoDB
Persistent expense storage with structured schema for fast querying.
